# Electronic Cash Machine (ATM)

Name: Prashant Adhikari
Course: DDOOCP

## Description
This is a GUI-based ATM system built using Java Swing. It allows users to enter PIN, check balance, withdraw money, and view last transaction.

## Features
- PIN authentication
- Balance enquiry
- Withdraw money
- Receipt system
- Insufficient funds protection
- ArrayList for account storage
- Inheritance (Savings & Current Account)
- Polymorphism

## How to Run
1. Compile all files:
   javac *.java

2. Run:
   java Main
