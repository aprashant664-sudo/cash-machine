## Description
GUI-based ATM system built using Java Swing.

## Features
- PIN authentication
- Balance enquiry
- Withdraw money
- Receipt system
- Insufficient funds check
- ArrayList used (Dynamic accounts)
- Inheritance (Savings & Current Account)
- Polymorphism

## How to Run
javac *.java  
java Main
